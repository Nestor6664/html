# SussexTravel Website

This is an entry-level HTML and CSS website project for SussexTravel.

## Project Structure

```
timurwork/
├── index.html          # Main HTML file
├── assets/
│   ├── css/
│   │   └── style.css   # All styling
│   └── images/         # Place images here
└── README.md           # This file
```

## Images Needed

You'll need to add these images to the `assets/images/` folder:

### Required Images:
1. **placeholder-destination.jpg** - Featured destination image (recommended size: 1200x300px)
2. **google-play.png** - Google Play Store badge (recommended size: 150x50px)
3. **app-store.png** - Apple App Store badge (recommended size: 150x50px)
4. **facebook-icon.png** - Facebook social media icon (50x50px, circular)
5. **twitter-icon.png** - Twitter/X social media icon (50x50px, circular)
6. **youtube-icon.png** - YouTube social media icon (50x50px, circular)
7. **instagram-icon.png** - Instagram social media icon (50x50px, circular)

## Color Scheme Used

- **Primary Blue**: #1a4d80 (Dark blue for header and main elements)
- **Secondary Blue**: #2d5f99 (Navigation bar)
- **Gold/Yellow**: #ffd700 (Accent color for buttons and highlights)
- **Orange**: #ff6b35 (Call-to-action buttons)
- **Dark Gray**: #2d2d2d (Footer background)
- **Light Gray**: #f5f5f5 (Page background)

## Features Implemented

✅ Fixed header and navigation (stays at top when scrolling)
✅ Responsive design for different screen sizes
✅ Navigation links with hover effects
✅ Footer links with hover effects (different from main nav)
✅ Social media icons with hover opacity effect
✅ Centered content with equal margins
✅ Semantic HTML5 tags where appropriate
✅ All styling in external CSS file

## How to View

1. Open `index.html` in your web browser
2. Make sure all images are in the `assets/images/` folder
3. The page will display correctly once images are added

## Notes for Learning

- The HTML uses semantic tags like `<header>`, `<nav>`, `<main>`, `<section>`, and `<footer>`
- CSS uses classes for styling (like `.header-content`, `.main-nav`)
- The layout uses CSS Grid and Flexbox for positioning
- Fixed positioning is used for header and navigation
- Hover effects are implemented using `:hover` pseudo-class
